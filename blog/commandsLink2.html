Comandos de la terminal

Comando Funcionalidad
ls Lista los archivos de la carpeta
cd Cambia de carpeta
clear Limpia la terminal
ls -l Lista todos los archivos del path
ls -a Muestra archivos ocultos
history Historial de comandos
cat (nombre del archivo) Muestra el contenido del archivo
mkdir (nombre) Crea una carpeta
touch (nombre) Crea un archivo
nano/vi (nombre del archivo) Editor de texto de la terminal
rm (nombre del archivo) Borrar archivo
cp (nombre de archivo a copiar) (nombre de carpeta a la que se va a copiar) Copia un archivo con el mismo contenido y nombre a otra carpeta
mv (nombre de archivo) (nombre de la carpeta) Para mover un archivo de una carpeta a otra
git add Añade un archivo al staging area
git status Muestra los archivos modificados y los que están en el staging area
git commit -m Para guardar los cambios hechos a un archivo en el repositorio
git push -u origin (branch) Para mandar el commit realizado al repositorio remoto
git pull Para jalar los cambios de un repositorio remoto a uno local
git fetch Para revisar los cambios de un repositorio remoto antes de hacerles pull
git switch Para cambiar de branch
git checkout -b Para crear una nueva branch
git merge Para juntar cambios de una rama a otra
