Fundamentos de CSS

¿Qué es CSS?

CSS (Cascading Style Sheets) o Hojas de Estilo en Cascada es un lenguaje que se utiliza para describir la presentación de documentos HTML. CSS define cómo se deben mostrar los elementos HTML en términos de layout, colores, fuentes y otros aspectos visuales. CSS permite separar el contenido (HTML) de su presentación, lo que facilita el mantenimiento y mejora la accesibilidad de nuestros sitios web.

Componentes principales de CSS

Selectores

Los selectores son patrones que se utilizan para seleccionar los elementos HTML que queremos estilizar. Identifican a qué elementos se aplicarán las reglas de estilo.

Propiedades

Las propiedades son características específicas de un elemento que podemos modificar, como color, font-size, margin, padding, etc.

Valores

Los valores son las configuraciones específicas que asignamos a cada propiedad. Por ejemplo, para la propiedad color, un valor podría ser red, #FF0000 o rgb(255,0,0).

Tipos de Selectores

Tipo Sintaxis Ejemplo Descripción
Selector de tipo elemento { } p { color: blue; } Selecciona todos los elementos del tipo especificado. En el ejemplo, selecciona todos los párrafos.
Selector de clase .clase { } .destacado { font-weight: bold; } Selecciona elementos con un atributo de clase específico. Se pueden aplicar a múltiples elementos.
Selector de ID #id { } #header { background-color: black; } Selecciona un elemento con un ID específico. Cada ID debe ser único en el documento.

Diferencias principales

  • Selector de tipo: Afecta a todos los elementos de ese tipo en la página. Es el menos específico.
  • Selector de clase: Permite reutilizar estilos en múltiples elementos. Tiene una especificidad media.
  • Selector de ID: Debe ser único en la página. Tiene la mayor especificidad, lo que significa que sus estilos prevalecen sobre los otros tipos.

La especificidad es importante en CSS ya que determina qué reglas tienen prioridad cuando varias reglas apuntan al mismo elemento. La jerarquía de especificidad (de mayor a menor) es: Estilos en línea > ID > Clase > Tipo.
