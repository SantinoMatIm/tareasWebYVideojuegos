Videogame API Documentation

Welcome to the Videogame API documentation. This API allows you to manage items and users for a videogame.

Endpoints

Method Endpoint Description
GET /items Get all items from the catalog
GET /items/:id Get a specific item by ID
POST /items Create one or more new items
PATCH /items/:id Update a specific item by ID
DELETE /items/:id Delete a specific item by ID
GET /users Get all users
GET /users/:id Get a specific user by ID
POST /users Create one or more new users
PATCH /users/:id Update a specific user by ID
DELETE /users/:id Delete a specific user by ID
GET / Get this API documentation page

Usage Examples

Getting all items

GET /items

Creating a new item

POST /items
Content-Type: application/json

{
  "id": 8,
  "name": "dragon scale",
  "type": "material",
  "effect": "+15 fire resistance"
}

Updating an item

PATCH /items/8
Content-Type: application/json

{
  "effect": "+20 fire resistance"
}

Deleting an item

DELETE /items/8
